Problema: Instalar uma impressora padrão que o Windows encontra automaticamente.
Solução: Primeiro, acesse as Configurações pressionando as teclas Win e I. Navegue até Dispositivos e depois Impressoras e scanners. Clique no botão para Adicionar uma impressora ou scanner e aguarde o Windows procurar pelos dispositivos. Se a sua impressora aparecer na lista, selecione-a e clique em Adicionar dispositivo. O Windows cuidará do resto. Ao finalizar, realize uma impressão de teste para confirmar a instalação.

Problema: O Windows não encontra minha impressora automaticamente durante a busca.
Solução: Após clicar em Adicionar uma impressora ou scanner e o Windows não encontrar seu dispositivo, clique na opção A impressora que eu quero não está na lista. Uma nova janela se abrirá com várias opções. Se sua impressora for mais antiga e estiver conectada via USB, escolha a opção Minha impressora é um pouco mais antiga para que o Windows faça uma busca mais profunda. Se a impressora estiver compartilhada em rede a partir de outro computador, escolha Selecionar uma impressora compartilhada pelo nome e insira o caminho de rede, como por exemplo, \NOME-DO-COMPUTADOR\NomeDaImpressora. Para impressoras de rede com um IP fixo, que é o método mais confiável em escritórios, escolha Adicionar uma impressora usando um endereço TCP/IP ou nome do host e insira o endereço IP do equipamento. Como último recurso, se você já baixou o driver do site do fabricante, escolha Adicionar uma impressora local ou de rede com configurações manuais para especificar a porta e o driver manualmente. Siga os passos do assistente para finalizar e teste a impressão.

Problema: Conectar um computador a uma impressora Epson (ex: L350) via Wi-Fi Direct, especialmente sem saber a senha.
Solução: Para começar, ative o modo Wi-Fi Direct na impressora pressionando o botão de Wi-Fi até que as luzes de status alternem continuamente, o que indica que a rede sem fio da impressora está ativa. Em seguida, no computador, procure pelas redes Wi-Fi disponíveis e conecte-se àquela com o nome similar a EPSON L350. Se a senha for solicitada, você pode obtê-la de duas formas. A primeira é verificando as propriedades da rede salva em um computador que já esteja conectado. A segunda é usando um computador conectado à impressora via cabo USB; neste computador, procure pelo aplicativo Epson Software Updater, ou baixe-o se não estiver instalado. Abra o aplicativo, selecione a impressora e, nas configurações de rede, haverá uma opção para tornar a senha visível. Anote a senha e use-a para conectar o novo computador.

Problema: O computador está conectado à rede, mas exibe a mensagem "Sem acesso à Internet" ou "Rede não identificada".
Solução: Primeiro, realize verificações básicas, como conferir se o cabo de rede está firmemente conectado em ambas as pontas e se não aparenta danos. Verifique também com o administrador da rede se há filtro de endereço MAC ativo, o que exigiria a liberação do seu dispositivo. Se o problema persistir, tente uma solução rápida reiniciando o adaptador de rede. Para isso, pressione Win e R, digite ncpa.cpl e pressione Enter. Na janela de conexões, clique com o botão direito no seu adaptador (Ethernet ou Wi-Fi), escolha Desativar, aguarde um momento e depois escolha Ativar. Se ainda assim não funcionar, o problema pode ser a atribuição de IP. Na mesma janela do ncpa.cpl, clique com o botão direito no adaptador, vá em Propriedades, selecione Protocolo IP Versão 4 (TCP/IPv4) e clique em Propriedades. Para entender a faixa da sua rede, verifique os detalhes de status da conexão e anote o Gateway Padrão IPv4. Na janela de propriedades do IPv4, marque a opção para Usar o seguinte endereço IP e preencha os campos. Para Endereço IP, use um IP como 192.168.X.21, substituindo o X pela faixa do seu gateway. A Máscara de sub-rede geralmente é 255.255.255.0. O Gateway padrão é o que você anotou. Abaixo, marque para Usar os seguintes endereços de servidor DNS e insira 8.8.8.8 como preferencial e 1.1.1.1 como alternativo. Clique em OK e reinicie o computador. Se o problema ainda continuar, a causa pode ser hardware ou drivers, exigindo uma atualização de driver pelo Gerenciador de Dispositivos ou a manutenção da placa de rede.

Problema: Acessar uma pasta compartilhada na rede e criar um atalho permanente para ela.
Solução: Primeiro, abra o Explorador de Arquivos. Na barra de endereço superior, digite o caminho completo da pasta de rede, como por exemplo, \servidor-principal\documentos, e pressione Enter. Se o sistema solicitar credenciais, insira o nome de usuário e a senha do computador que está compartilhando a pasta e, se desejar, marque a caixa para Lembrar minhas credenciais para não ter que digitar novamente no futuro. Para criar um atalho permanente, conhecido como mapeamento, clique com o botão direito em Este Computador, escolha a opção Mapear unidade de rede, e siga as instruções para atribuir uma letra de unidade, como Z:, à pasta de rede.
